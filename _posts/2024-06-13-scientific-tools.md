Este post irá listar algumas ferramentas facilitadoras no trabalho dos pequisadores de uma maneira geral, mas como enfoque para a área de exatas.

Segue lista de cada uma das ferramentas com link para acesso:


- Gerenciadores de referências: *Mendeley* e *Zotero*;
- Pre-prints: *Arxiv* e *Pre-prints*;
- Ferramentas para busca de artigos: *Scopus*, *WebOfScience*, *Mendeley*, *IEEEexplore*; 
- Sistema de armazenamento e publicação de produtos de pesquisa científica: *Zenodo*;
- Cadastro de Currículo Acadêmico: *OrCID* e *Lattes*;
- Latex: *Overleaf*, *MathPix*;
- Repositórios de Código: *Github*, *Gitlab*.
- Sistemas de gerenciamento de notas: *Obsidian*, *LogSeq*, *Notion*, *Workflowy*;
